졸업과제 - 신약 개발 후보물질 발굴을 위한 거대 화합물 라이브러리 탐색 최적화

2024.05 ~ 2024.10

팀 프로젝트(3인)

  • Python
  • Django
  • HTML
  • CSS
  • JavaScript

소개

신약 개발 후보물질 발굴을 위한 거대 화합물 라이브러리 탐색 최적화를 주제로 프로젝트를 진행하고, 그 결과를 웹 애플리케이션의 형태로 시각화

기능
- 라이트 모드/다크 모드
- 리간드 데이터베이스에 추가
- 리셉터 파일을 업로드 및 탐색(Random, Clustering, MEMES)
- 미리 업로드 된 리셉터를 이용한 데모 - 3Dmol.js를 이용한 도킹 포즈 시각화

역할
- rdkit을 이용한 clustering 구현
- MO-MEMES 소스코드 수정 및 MEMES 알고리즘 테스트
- 웹 애플리케이션 UI 디자인 및 구현
- MEMES에 Acquisition Function 추가 구현

* 보고서는 일부만 수록되었습니다.
